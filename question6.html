Difference Between HTML and HTML:5

HTML HTML:5
Requires lengthy doctype decelaration Requires short doctype decelaration
It lacks many sementic tags It has many semantic tags like <nav>, <table> etc.
Under HTML we have to relay on plugins such as adobe flash player Under HTML:5 we don't need to use external plugins
Under HTML we get limited types of input for forms We get large number of input type for the forms.
Under HTML we don't get any support of APIs. Under HTML:5 we get support of APIs.
