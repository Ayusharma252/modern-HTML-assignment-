Q1. List out featuers of Html 5?

Features of HTML:5 are:-

  1. Sementic Elements
  2. Audio Video Support
  3. Local Storage
  4. Responsive Images
  5. Form Enhanchments
  6. Drag and Drop API

1. Sementic Elements

By implementing the sementic elements now it has became more clear which tag will hold what information like <div> don't shows what is stood inside it but <P> tag tells us the paragraph has been placed inside it.

2. Audio and Video Support

Implementation of Audio and video support has reduced the reliablity on the third party services such as Adobe flash player.

3. Local storage

Now, we can store data in the user's browser which can be acessed with the help of javascript APIs.this helps to reduce the transcript between application and backend server which helps to make the application fast.

4. Responsive image

The implementaion of srcset attribute helps us to specify multiple versions of an image at different resolutions. Earlier for the same process we had to be depended on css and javascript.

Form Enhancement

The new enhancement in the HTML:5 include new input types such as email, URL nad more, Placeholder text, Required feilds featme, validation nad more

6. Drag and Drop API

Drag and drop allows us tpo grab any elements in DOM and drop it to different location. to create an element able to drag and drop we just set the attribute to "Draggable" and on the tag we put its value to true.

Q3. Define accessibility in the context of web development. Define why it's essential to create a accessible website and how it benifits different user groups.

In context of web development accesibility can be defined as making web accessible to disabled. People like the person people like the person who can't see overall we can say that accesibility refers to make website and technologies available to every person.

Reasons to make a acccessible website are mentioned below.

  1. Many countries have made laws to make website accessible
  2. Making website accesible ensures that all persons including persons with disability have access to web content.
  3. Making accesible websites enhances the expoerience of the website.
  4. Makes the website reach to a larger audience, which increase websites traffic and engagement.
  5. A good website enhances brand repution.
  6. Designing a accessible website is a Ethical resposibility.It ensures that no-people is excluded from the digital world.

Accesible website helps different group of people in following ways.

  1. Making screen reader version of website helps people with visual imapairments
  2. Providing Captions of audio and video helps content helps people who are deaf or hard of hearing.
  3. Voice command helps people with motor impairments.
  4. Make website accessible with scalable text, high contrast, captions etc helps older adults.
  5. Optimized, lightweight content and alternative text for image enhances performance of website for slower internet connection which is a common problem with general population.

Q4. List any 3 ways which helps us in improving the accessibility of HTML?

These are the 3 ways which enhances the accesibility of webpage:-

  1. By using sementic tags: By using sementic tag we can provide built in functions of web-browers like screen Reader and keyboard navigation
  2. Providing Alt text for image: Providing ALT tag for image helps when the net is slow and also the ALT text is read by screen reader
  3. Choosing accessible colours: We must ensure that the colours we are using have good contrast and texts are redable. This helps the people with visula impairments.
